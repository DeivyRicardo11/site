WINDOWS

Windows 1.0

O Windows 1.0 é o primeiro grande lançamento do Microsoft Windows, uma família de sistemas operacionais gráficos para computadores pessoais desenvolvidos pela Microsoft. Foi lançado pela primeira vez para fabricação nos Estados Unidos em 20 de novembro de 1985, enquanto a versão europeia foi lançada como Windows 1.02 em maio de 1986.

Windows 2.0

O Windows 2.0 é uma versão principal do Microsoft Windows, uma família de sistemas operacionais gráficos para computadores pessoais desenvolvida pela Microsoft. Foi lançado para fabricação em 9 de dezembro de 1987, como sucessor do Windows 1.0.

Windows 3.0

O Windows 3.0 é o terceiro grande lançamento do Microsoft Windows, lançado em 22 de maio de 1990. Ele introduz uma nova interface gráfica de usuário (GUI) que representa aplicativos como ícones clicáveis, em vez da lista de nomes de arquivos em seus predecessores. Atualizações posteriores expandem recursos, como suporte multimídia para gravação e reprodução de som e suporte para CD-ROMs .

Windows 4.0

O Windows NT 4.0 é uma versão principal do sistema operacional Windows NT desenvolvido pela Microsoft e voltado para empresas. É o sucessor direto do Windows NT 3.51, e foi lançado para fabricação em 31 de julho de 1996, e depois para varejo em 24 de agosto de 1996, com as versões Server lançadas para varejo em setembro de 1996.

Windows 5.0

O Windows CE 5.0 (codinome "Macallan"), é um sucessor do Windows CE 4.2, o terceiro lançamento da família Windows CE NET. Foi lançado pela primeira vez em 9 de julho de 2004. Como seus predecessores, o Windows CE 5.0 é comercializado para o mercado de dispositivos embarcados e fornecedores de dispositivos independentes. O Windows CE 5.0 é anunciado como um sistema operacional de baixo custo, compacto, rápido para o mercado e em tempo real, disponível para sistemas baseados em microprocessadores x86, ARM, MIPS e SuperH.

Windows 6.0

O Windows Embedded CE 6.0 (codinome " Yamazaki "), é o sexto grande lançamento do sistema operacional Microsoft Windows Embedded, voltado para ferramentas específicas de empresas, como controladores industriais e dispositivos eletrônicos de consumo, como câmeras digitais. O CE 6.0 apresenta um kernel que suporta 32.768 processos, acima do limite de 32 processos das versões anteriores. Cada processo recebe 2 GB de espaço de endereço virtual, acima dos 32 MB. O Windows Embedded CE é comumente usado em caixas de autoatendimento de supermercados e carros como um display. O Windows Embedded CE é um sistema de segundo plano na maioria dos dispositivos que o possuem.

Windows 7.0

O Windows 7 é uma versão principal do sistema operacional Windows NT desenvolvido pela Microsoft. Foi lançado para fabricação em 22 de julho de 2009 e tornou-se disponível em geral em 22 de outubro de 2009. É o sucessor do Windows Vista, lançado quase três anos antes. A versão de servidor do Windows 7, o Windows Server 2008 R2, foi lançada ao mesmo tempo. Foi sucedido pelo Windows 8 em outubro de 2012.

Windows 8.0

O Windows 8 foi uma versão do Microsoft Windows, uma série de sistemas operacionais desenvolvidos pela Microsoft para computadores pessoais, laptops e tablets. É o sucessor do Windows 7. Foi anunciado oficialmente por Steve Ballmer, diretor executivo da Microsoft, durante a conferência de pré-lançamento do sistema operacional. O Windows 8 Developer Preview, primeira versão beta do sistema, foi lançado no dia 13 de setembro de 2011 e depois foi seguida pela versão Consumer Preview no dia 29 de fevereiro de 2012. No dia 31 de maio de 2012, foi liberada para download a versão Windows 8 Release Preview. A versão final foi lançada mundialmente em 26 de outubro de 2012. O Windows 8 foi substituído pelo Windows 8.1.

Windows 9.0

O Windows 9 é o novo sistema operacional da Mic esperado como Windows 9, o SO vem para unificar plataformas e rodar em todos os devices, de celulares a tablet e de híbridos a computadores desktop. A proposta do Windows 10 é oferecer ao usuário uma abordagem mais simples e amigável de separar dados corporativos e pessoais em todos os dispositivos, mantendo acesso a tudo o que precisa na hora certa. A ferramenta é compatível com todos os softwares de gestão tradicionais, com mais foco empresarial do que nunca.

Windows 10.0

As atualizações de recursos do Windows 10 serão lançadas anualmente, na segunda metade do ano civil, no Canal de Disponibilidade Geral. Eles receberão atualizações de qualidade mensais por 18 ou 30 meses a partir da data de lançamento, dependendo da política de ciclo de vida. Recomendamos que você inicie a implantação de cada lançamento do Canal de Disponibilidade Geral imediatamente como uma implantação direcionada para dispositivos selecionados para adoção antecipada e que realize a implantação completa a seu critério. Isso permitirá que você obtenha acesso a novos recursos, experiências e segurança integrada assim que possível.

Windows 11.0

O Windows 11, é um sistema operacional e a versão atual do sistema Microsoft Windows (série de sistemas operacionais para computadores pessoais, laptops e tablets) desenvolvido pela Microsoft, anunciada em 31 de agosto de 2021 e lançado em 5 de outubro de 2021, como o sucessor da versão Windows 10 (lançado seis anos antes em 2015).

Windows 12.0

O Windows 12 deve ser o próximo sistema operacional da Microsoft. Mesmo que não haja nenhum comentário oficial por parte da big tech, já existem inúmeros rumores na Internet sobre a próxima versão da tecnologia. Sites como PCMag e Software Keep especulam, inclusive, que o software pode ser lançado ainda em 2024. Por outro lado, outros rumores apontam que o Windows 11 será a última versão do sistema e que, a partir de agora, serão lançadas apenas atualizações contínuas. Em meio a essa incerteza, veja a seguir tudo que se sabe sobre o Windows 12.

Windows 13.0

Windows 13 O ícone do Windows 13 (codinome Palisade) foi o décimo terceiro grande lançamento do Windows. O Windows 13 pretendia ser uma reformulação total do próprio Windows. O Windows 13 tinha um kernel totalmente novo chamado Quantum Kernel, destinado a preparar o Windows para máquinas quânticas e a ser usado como um kernel universal, permitindo que o Windows fosse executado em qualquer dispositivo. Além disso, abandonou o modelo “Windows como serviço”, o que levou a Microsoft a voltar a lançar uma nova versão do Windows a cada 3-5 anos.

              -Distribuições-

Linux ArchLinux

É uma distribuição não comercial e livre de Linux. O usuário tem liberdade para definir o que vai ser instalado no sistema e os pacotes são disponibilizados como foram liberados pelos desenvolvedores originais, sem qualquer alteração. A palavra “arch” em inglês significa “arco” e reflete a ideia que o objetivo da distribuição é ser apenas a estrutura de apoio dos softwares desenvolvidos por outros autores.

Linux Clear

É uma distribuição Rolling Release. O projeto é focado no hardware Intel (tudo o que for da Intel estará altamente otimizado a fim de garantir um excelente desempenho) e no cenário de um ambiente de nuvem com muitas máquinas.

Linux Debian

É um sistema operacional de distribuição não comercial e livre que usa o kernel Linux ou kfreebsd (kernel do FreeBSD). Como o Debian se baseia no projeto GNU, normalmente é chamado de Debian GNU/Linux ou Debian Gnu/kFreeBSD. Atualmente várias distribuições comerciais se baseiam no Debian como, por exemplo, Kurumin e Ubuntu. O nome “Debian” vem da junção do nome do principal fundador, Ian, com o de sua esposa, Debra.

Linux Deeph

É uma distribuição baseada no Debian. Apresenta uma interface própria chamada Deepin Desktop Environment (DDE), além de uma acervo de aplicações próprias. É desenvolvida pela Wuhan Deepin Technology Company Limited, localizada em Wuhan, China.

Linux Gentoo

É um sistema operacional livre que pode tanto ser baseado em Linux quanto em FreeBSD. Usa uma tecnologia chamada Portage que possibilita ao usuário customizar o sistema como quiser (ele escolhe os pacotes que irão fazer parte do sistema operacional). “Gentoo” é nome de uma das espécies de pinguim da Antártida.

Linux Fedora

É uma distribuição não comercial do Linux voltada para desktop e patrocinada pela Red Hat Enterprise Linux (dona da marca Fedora). Ela nasceu quando a empresa norte-americana decidiu investir em uma versão comercial do Linux, o Red Hat. O nome “Fedora” corresponde ao tipo de chapéu que aparece no logotipo da Red Hat.

Linux Manjaro

É uma distribuição baseada no Arch Linux e voltada principalmente para computadores pessoais, mas conta com o seu próprio conjunto de repositórios. Por ser uma distribuição Rolling Release, basta digitar “sudo pacman -Syu” no terminal para se usar a última versão.

Linux Kali

É uma distribuição não comercial voltada para os profissionais da área de segurança da informação. Ela disponibiliza várias ferramentas para detectar falhas de segurança que permitem a invasão de um sistema, por isso ela é definida como Penetration Testing Distribution. A distribuição Kali substituiu a distribuição backtrack que foi descontinuada (2013). Kali é o nome de uma deusa hindu com força destrutiva e significa “the black one” em sânscrito.

Linux Red Hat

É uma distribuição comercial do Linux voltada para clientes corporativos. Foi a primeira distribuição linux a usar um sistema de gerenciamento de pacotes. O nome “Red Hat” é uma referênciais ao boné vermelho do time de Lacrosse da Universidade Cornell dado ao fundador da companhia Marc Ewing por seu avô.

Linux Slackware

É uma distribuição não comercial que tem por objetivo fornecer um ambiente o mais próximo possível do ambiente Unix. Por isso, faz uso de arquivos de texto e de shell scripts para configuração e administração do sistema. O nome “Slackware” pode ser traduzido por “cuidado com a escória” e é uma referência à Igreja de Subgenius (Church of SubGenius), uma organização que ironiza as religiões conhecidas.

Linux Solus

"

É um sistema operacional independente baseado no kernel Linux e que usa o conceito de Rolling Release. O ambiente gráfico é o Budgie Desktop que disponibiliza diversos recursos como permitir a exibição de “widgets” como o calendário, player de música, controle de saída e entrada de áudio, etc.

Linux SUSE

É uma distribuição comercial com versões para empresa, servidor, computação em nuvem e virtualização. O nome “SUSE” vem de um acrônimo em língua alemã para “Software und System-Entwicklung” e que significa “Software e Desenvolvimento de Sistemas”.
