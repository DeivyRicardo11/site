1.1 Componentes Internos

1.1.1 CPU

Unidade central de processamento, responsável pelo processamento de dados e execução de instruções.

1.1.2 Armazenamento

Dispositivos como HDDs e SSDs utilizados para armazenar dados permanentemente.

1.1.3 Memória

Componente responsável pelo armazenamento temporário de informações para rápido acesso.

1.1.4 Fonte de Alimentação

Responsável por converter a energia elétrica da rede em tensão adequada para os componentes internos do computador.

1.2 Componentes Externos

1.2.1 Teclado

Dispositivo de entrada utilizado para inserir caracteres e comandos.

1.2.2 Monitor de vídeo

Dispositivo de saída que exibe informações visuais para o usuário.

1.2.3 Mouse

Dispositivo apontador utilizado para interagir com interfaces gráficas.

2. Grandezas Elétricas

2.1 Potência elétrica

Refere-se à taxa de transferência de energia elétrica.

2.2 Tensão elétrica

A diferença de potencial elétrico entre dois pontos.

2.3 Corrente elétrica

Fluxo de carga elétrica através de um condutor.
