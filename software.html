Conceito de Software

O que é um sistema operacional?

Um sistema operacional (SO) é um software fundamental que gerencia o hardware e os recursos de um computador. Ele atua como uma ponte entre o usuário e o hardware, garantindo que os recursos sejam utilizados de forma eficiente.

Funções principais:

  • Gerenciamento de Processos: Controla os programas em execução e aloca tempo de CPU.
  • Gerenciamento de Memória: Organiza o uso da RAM pelos aplicativos.
  • Gerenciamento de Armazenamento: Controla os dados no disco rígido.
  • Gerenciamento de Dispositivos: Coordena a comunicação com periféricos.
  • Interface com o Usuário: Oferece GUI ou CLI para interação.

Exemplos de Sistemas Operacionais:

  • Windows (Microsoft)
  • Linux (Ubuntu, Fedora, Debian)
  • macOS (Apple)
  • Android e iOS

Sistemas Livres

Software livre é um software que permite ao usuário executar, acessar, modificar e redistribuir seu código-fonte sem restrições. Foi criado por Richard Stallman e definido pela Free Software Foundation.

Exemplos:

  • Sistema Operacional Linux
  • Linguagens Java e PHP
  • Banco de Dados MySQL

Sistemas Proprietários

Diferente do software livre, os softwares proprietários possuem código-fonte fechado e restrições de uso impostas pelos desenvolvedores.

Exemplos:

  • Adobe Photoshop
  • Suíte Microsoft Office
  • Apple macOS

Softwares de Sistemas

Softwares de sistemas são aqueles que oferecem uma plataforma para o funcionamento de outros softwares, como sistemas operacionais e drivers.

Exemplos:

  • macOS
  • iOS
  • Android

Softwares Aplicativos

Softwares aplicativos são programas desenvolvidos para realizar tarefas específicas para os usuários, como editores de texto, navegadores e reprodutores de mídia.

Exemplos:

  • Microsoft Office
  • Google Chrome
  • Adobe Photoshop
